Normal Map Inspector - Krita Client extension

Normal Map Inspector is a tool for pixelart that makes drawing normal maps easy.

This extension allows to connect Krita with the NM Inspector app, so you can preview your normal maps while you edit them.

Usage

  1. Start the NM Inspector app.
  2. Open a document in Krita. First you can check out the template included in the NM Inspector package.
  3. Click Tools -> Scripts -> NM Inspector Client.
  4. Select layers you want to inspect.
  5. Click Connect.
For more details, please visit https://ppelikan.itch.io/normal-map-inspector
